### 📝 Project Description
LM35 analog temperature sensor use chesi real-time temperature ni 16x2 LCD meedha display chesina Embedded Systems mini project. 

ADC module use chesi LM35 nunchi vache analog voltage ni digital value ga marchi, C programming tho Celsius lo temperature calculate chesanu.

### 🔧 Hardware Used
- LM35 Temperature Sensor
- Arduino Uno / ATmega328P Microcontroller 
- 16x2 LCD Display
- Breadboard & Jumper Wires
- 5V Power Supply

### 💻 Software & Tools

- Embedded C Programming
- Proteus for Simulation
